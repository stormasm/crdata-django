Data

To upload data click Upload or on Upload above.

Currently, all input data must be in tabular format and saved in Comma Separated Values (CSV) file format. CSV format files are plain text and can be easily generated with text editors, Microsoft Excel, Open Office, Google Docs and Spreadsheets and other spreadsheet programs. The first row and column in CRdata input files are assumed to be row and column labels (names).

The View menu item takes you to a listing of the files you have uploaded and allows you to view the contents of the individual data files.

The Delete menu item allows you to delete data files that you have already uploaded.
